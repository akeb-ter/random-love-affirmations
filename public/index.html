What is this?

A simple JSON API that returns love affirmations. Each response includes lang and affirmation. Lightweight, fast, and easy to integrate.

API Endpoints

Get the full dataset

/api

Get one random affirmation

/api/random

Random affirmation by language/dialect

/api/random?lang=english /api/random?lang=filipino /api/random?lang=bisaya /api/random?lang=ilonggo
